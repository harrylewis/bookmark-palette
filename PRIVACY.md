# Bookmark Palette — Privacy Policy

*Last updated: April 19, 2026*

## Summary

Bookmark Palette does not collect, store, or transmit any data. It makes no
network requests. Everything happens locally on your device.

## Permissions

The extension requires the following Chrome permissions:

**Bookmarks** — to read your bookmark folders and display them in the palette.
Your bookmarks are read locally and never leave your device.

**Tabs** — to open new tabs when you select a bookmark folder.

**Tab Groups** — to group the opened tabs together and assign them a name
and colour.

**Active Tab** — to inject the palette UI into the current tab when you
trigger the extension via keyboard shortcut or toolbar button. Access is
granted only at the moment of invocation and does not persist.

**Scripting** — required to programmatically inject the palette UI into the
active tab on demand. No scripts are injected without an explicit user
action.

## Data storage

The extension stores nothing. No browsing history, no bookmark data, no usage
statistics, no identifiers of any kind are written to local storage or
transmitted anywhere.

## Network requests

The extension makes no network requests. All fonts and assets are bundled
locally within the extension package.

## Third-party code

The extension bundles the following open source fonts, which are loaded
locally and make no external requests:

- DM Sans — licensed under the SIL Open Font License 1.1
- Roboto Mono — licensed under the Apache License 2.0
